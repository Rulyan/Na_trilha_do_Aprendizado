HTML

É uma Linguagem de Marcação de Hipertexto baseada em tags criadas na década de 1990 por Tim Berners-Lee.

A HTML foi criada para publicação de conteúdo em hipertexto, tais como:

  • texto;
  • imagem;
  • áudio
  • vídeo
Estes elementos podem ser vinculados através de hyperlinks.

TAGS

• As tags definem ao navegadora marcação de conteúdo.

• Tem funcionalidades específicas. Sem o use de tags o navegador exibe um texto plano.

• Podem possuir atributos, ou seja, são parametros que definem algum tipo de informação extra ao navegador.

Formatos de imagens

-São padrões de codificação, compressão, e definição que determina o tipo de arquivo gráfico, como exemplo temos:

• .BMP[Windows Bitmap]
O formato é basicamente um arquivo que descreve quantos pixels a imagem tem e qual é a cor de cada pixel. Os valores de cada luz variam de 0 a 255 porque cada um deles têm 8 bites (ou um byte).
Ou seja, cada pixel de uma imagem BMP tem três bytes.

• .TIFF[Tagged Image Format]
-É usado extensamente em aplicações de manipulação de imagens tais como Photoshop, DTP e scanners.

• .GIF[Graphics Interchange Format]
Surgiu antes do PNG, com a intenção de suprir a deficiência da compressão JPG para arquivos gráficos. É o melhor formato para imagens em movimento

• .SVG[Scalable Vector Graphics]
-É um arquivo vetorial criado especialmente para a web. Funciona tanto com imagens estáticas, quanto com animações em 2D.

• .ICO[Icon]
Um arquivo ICO é um formato de imagem usado para armazenar ícones de programas, arquivos e pastas do Windows. Ele contém dois bitmaps: 1) o bitmap AND - a máscara de imagem (que determina qual parte do ícone é transparente) e
2) o bitmap XOR - contém o ícone que é mapeado na máscara de imagem.

• .JPG[Joint Photographics Experts Group]
Diferente do formato BMP, ele divide a imagem em blocos de 8 por oito pixels e compara cada um a um dos 64 padrões existentes. Em seguida, determina qual o “peso” de cada um desses padrões em cada bloco.

• .PNG[Portable Networks Graphics]
-Criado para facilitar a troca de imagens pela internet, assim como o JPG, esse formato usa compressão para reduzir o peso de arquivos.
O formato reúne características do JPG e do GIF, porém com opções mais avançadas. Não preserva o fundo.

• .GIF[Graphics Interchange Format].EPS
-Pode ser usado para armazenar gráficos. Formato utilizado principalmente quando há recortes ou traçados, além de seu uso para imagens vetoriais e bitmap.

Hiperlinks:

É um referência dentro de um só mesmo documento, outros documentos, arquivos ou endereço eletrônico.

Os links podem ser remotos, iternos, locais, com endereços de correio ou arquivos.

Marcação de Hiperlinks:

Utiliza a tag "< a >" e a propriedade "href" que faz a referência para ao conteúdo desejado.

Sintaxe:

Hiperliks Externos ou Remotos:

• São liks que referenciam sites ou págias da web. Na propriedade "href" utilizamos a URL(Uniform Resource Locator) para a referência que desejamos.

Sintaxe:
< a href = "URL" > Conteúdo Referenciado < /a >

• Com a propriedade "target" você pode escolher entre abrir o link nna própria página com "_self" ou abrir em outra guia com "_blank".

Hiperlinks Internos:

• São links que apontam a um lugar diferente dentro da mesma págia e possibilita acesso à diferentes parágrafos ou seções quando o documento é extenso.
Sintaxe:

Hiperlinks Locais:

• São liks que apontam para outras páginas detro de um mesmo site.

Softwares de Versionamento

Sistema que tem por objetivo realizar o controle de versões de um código ou documento. É possível gerenciar as versões de
um código ou documento realizadas por um desenvolvedor ou equipe de desenvolvedores.

Git

• É um software livre de versionamento que surgiu para o desenvolvimento do Kernel do linux, foi projetado e desenvolvido por Linus Torvalds em 2005.
Cada diretório de trabalho do Git é um repositório com um histórico completo e habilidade total de acompanhamento das revisões , não dependentede acesso a uma rede ou a um servidor central.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

GitHub

• É uma plataforma de hospedagem de código-fonte e arquivos com controle de versão. Surgiu em 2008 e foi adquirido em 2018 pelo Microsoft por US$ 7,5 bilhões.
• Mais ou menos 100 milhões de projetos;
• E mais de 56 milhões de usuários.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Comandos Git:

• Comit:
Torna permanente os conjuntos de alterações em um repositório.
• Push:
Envia alterações para o servidor no repositório remoto.
• Fetch:
Verifica de o seu repositório local está em sincronia com o remoto.
• Pull:
Sincroniza o repositório local com o remoto, atualizando as versões.
